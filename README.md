## Comandos

- Creación del proyecto

  ```txt
  npm init -y
  ```

- Reinstalar o recuperar los `node_modules`
  
  ```txt
  npm install
  ```

- Transpilar el proyecto de TS a JS (2 opciones)
  
  ```txt
  npm run build
  ```

  ```txt
  tsc --watch
  ```

- Desplegar el servidor (2 opciones en producción, 1 opción en desarrollo)
  
  ```txt
  npm run dev
  ```
  
  ```txt
  node dist/index.ts
  ```
  
  ```txt
  nodemon dist/index.ts
  ```
